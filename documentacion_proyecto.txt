================================================================================
                              BASE DE DATOS
================================================================================

Nombre: apbarcelona

Tabla: usuarios
- id (INT, PRIMARY KEY, AUTO_INCREMENT)
- nombre (VARCHAR 100)
- email (VARCHAR 150, UNIQUE)
- password (VARCHAR 255)
- telefono (VARCHAR 20)
- created_at (DATETIME)
- updated_at (DATETIME)

Tabla: apartamentos
- id (INT, PRIMARY KEY, AUTO_INCREMENT)
- user_id (INT, FOREIGN KEY -> usuarios.id)
- nombre (VARCHAR 100)
- email (VARCHAR 150)
- telefono (VARCHAR 20)
- barrio (VARCHAR 50)
- habitaciones (VARCHAR 10)
- precio (DECIMAL 10,2)
- descripcion (TEXT)
- direccion (VARCHAR 255)
- estado (ENUM: pendiente, aprobado, rechazado)
- created_at (DATETIME)

================================================================================
                           1. INTRODUCCIÓ
================================================================================

El present document correspon al projecte intermodular desenvolupat com a
treball final del curs 2025-2026 del Cicle Formatiu de Desenvolupament
d'Aplicacions Web (DAW).

El projecte consisteix en una plataforma web per a la gestió i reserva
d'apartaments turístics a Barcelona, anomenada ApBarcelona.

L objectiu principal és crear un aplicatiu complet que permeti:

- Publicar apartaments turístics
- Reservar apartaments
- Gestionar reserves
- Autenticació d'usuaris

================================================================================
                        2. OBJECTIUS DEL PROJECTE
================================================================================

Objectius principals:
- Crear una interfície web responsive i accessible
- Implementar sistema de reserves en línia
- Gestionar usuaris (registre, login)
- Publicar catàleg d'apartaments
- Sistema de reserves funcional

Objectius tècnics:
- Ús de PHP backend
- Base de dades MySQL
- Disseny amb CSS modern
- JavaScript per a interactivitat

================================================================================
                       3. REQUISITS TÈCNICS

                         5. BASE DE DADES

Taula: usuarios
- id (INT, PK, AI)
- nombre (VARCHAR 100)
- email (VARCHAR 150, UNIQUE)

- telefono (VARCHAR 20)
